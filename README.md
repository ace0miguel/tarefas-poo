## ▶️ Como compilar e executar (Java)

Entre na pasta da tarefa desejada e execute:

```bash
javac src/*.java -d out
java -cp out Main
```

## 📌 Convenções

- Cada tarefa fica em sua própria pasta (`tarefa01/`, `tarefa02/`, etc.)
- O código-fonte fica em `tarefaNNN/src/`
- Os arquivos compilados ficam em `tarefaNNN/out/` (ignorados pelo Git)
