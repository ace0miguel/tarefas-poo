# Tarefa 01 — Introdução à Orientação a Objetos

## Descrição

Criar uma classe `Pessoa` com os atributos `nome` e `idade`, um construtor, métodos getters e um método `apresentar()` que exibe os dados da pessoa.

Em seguida, criar a classe `Main` para instanciar objetos `Pessoa` e demonstrar o uso da classe.

## Requisitos

- [ ] Classe `Pessoa` com atributos privados `nome` (String) e `idade` (int)
- [ ] Construtor com parâmetros `nome` e `idade`
- [ ] Métodos `getNome()` e `getIdade()`
- [ ] Método `apresentar()` que imprime os dados da pessoa
- [ ] Classe `Main` com pelo menos dois objetos `Pessoa`

## Como executar

```bash
javac src/*.java -d out
java -cp out Main
```
